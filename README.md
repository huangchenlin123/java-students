# 📚 学生信息管理系统 (Student Management System)

基于 **Spring Boot + MyBatis + MySQL** 的学生信息管理系统，提供 RESTful API 实现学生信息的增删改查功能。

---

## 🚀 技术栈

| 技术 | 版本 | 说明 |
| :--- | :--- | :--- |
| **Spring Boot** | 3.0.5 | Web 框架 |
| **MyBatis** | 3.0.3 | ORM 框架 |
| **MySQL** | 8.0.33 | 数据库 |
| **Druid** | 1.2.25 | 数据库连接池 |
| **Lombok** | 1.18.46 | 简化 Java 代码 |
| **JDK** | 21 | Java 运行环境 |
